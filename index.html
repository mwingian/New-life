Про нас

Фонд “Życie od nowa ” працює на благо мешканців Кракова.



Організуємо зустрічі, психологічні консультації, психотерапевтичні сеанси, психологічну допомогу біженцям, організацію педологічної групи підтримки підлітків та дітей, організацію психологічної гарячої лінії, арт-терапію для психологів, супервізію для психологів, індивідуальну терапію, терапію для пар, сімейну терапію, консультації для батьків, психологічна та фізіотерапевтична реабілітація.



Допомагаємо при виході з депресії, робота з самооцінкою, корекція поведінки, набуття уміння взаємодіяти в колективі та ін, проведення тренінгів, тренінгів, семінарів та конференцій. Організація зустрічей та консультацій з психологами, психіатрами та фізіотерапевтами, а також при необхідності з іншими фахівцями.
